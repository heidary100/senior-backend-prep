# Senior Backend Engineer Prep

This repository is a structured study guide and hands-on workspace for preparing for **senior backend/NestJS interviews**.  
It covers:

- NestJS internals and advanced TypeScript
- Architecture patterns (DDD, CQRS, microservices)
- Database optimization
- Security best practices
- Performance and scalability
- Testing strategies
- DevOps and production readiness
- System design exercises
- Interview questions (behavioral & technical)

## How to Use

1. Read theory files (Markdown)
2. Run hands-on examples
3. Take notes & update repo with your own insights
4. Practice system design and coding exercises
